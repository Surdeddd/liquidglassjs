Living spring physics: mass-spring-damper system drives press-squash, release-wobble and hover-magnetism on any backend via composed host transforms. Enabled by default, configurable through the `physics` option (press/hover/wobble), fully disabled by `physics: false`, the `physics="false"` attribute or prefers-reduced-motion. The rAF loop runs only while springs are moving.
